# 🩺 EstomaCare

**Sistema de documentação de curativos e estomaterapia**  
Complexo Hospitalar Padre Bento de Guarulhos — CHPBG

---

## Como publicar no GitHub Pages (gratuito)

### Passo 1 — Criar conta no GitHub
1. Acesse **github.com** e crie uma conta gratuita
2. Confirme o e-mail

### Passo 2 — Criar repositório
1. Clique em **"New repository"** (botão verde)
2. Nome: `estomacare` (tudo minúsculo)
3. Marque **"Public"**
4. Clique em **"Create repository"**

### Passo 3 — Enviar os arquivos
No repositório criado, clique em **"uploading an existing file"** e envie **todos** estes arquivos:
- `index.html`
- `manifest.json`
- `sw.js`
- `icon-192.png`
- `icon-512.png`
- `icon-180.png`

Clique em **"Commit changes"** (botão verde).

### Passo 4 — Ativar GitHub Pages
1. Vá em **Settings** (engrenagem) → **Pages**
2. Em "Branch", selecione **main** e pasta **/ (root)**
3. Clique **Save**
4. Aguarde ~1 minuto

### Passo 5 — Seu link estará disponível em:
```
https://SEU_USUARIO.github.io/estomacare/
```

## Como instalar no celular

### Android (Chrome):
1. Abra o link acima no Chrome
2. Um banner aparece automaticamente: **"Instalar EstomaCare"**
3. Toque em **Instalar** → o ícone aparece na tela inicial

### iPhone (Safari):
1. Abra o link no Safari
2. Toque no ícone de **Compartilhar** (caixinha com seta)
3. Toque em **"Adicionar à Tela de Início"**
4. Confirme tocando em **"Adicionar"**
